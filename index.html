Projects

Highlights from My Work

PyPI
Stablecoins Dutch Auction On/Off-Ramp
  • Decentralized off-ramp platform that converts stablecoins to INR via UPI payments (UPI is India's instant, universal money-movement layer that lets anyone pay anyone, anywhere, in seconds, for free) under an average time of 20 seconds.
  • This off-ramp is inspired by One-Inch's "Fusion Plus Protocol", including third-party resolver bots and Dutch auction pricing mechanism which enables users to often get exchange price better than the market price.
Github Demo
Kinora
Solidity TEE AI IPFS
  • This platform allows users to tokenize themselves by minting and maintaining their own AI-backed persona NFTs.
  • These transferable assets fully personify their original minters and enable their holders to interact with them.
  • The users' memory bank was stored on IPFS using asymmetric-key encryption.
  • All the AI computations were executed inside Phala's TEEs, ensuring that no personal data of any user was exposed.
  • This project also implemented "Vickrey's second-price auction" fully on-chain, a mechanism named after Nobel laureate William Vickrey, who won the 1996 Nobel Prize in Economic Sciences.
Github Demo Videos Website
Warriors AI-rena / RANN
AI Agents VRF
  • A fully on-chain PvP battle game inspired by Pokemon, where players mint warrior NFTs of any character, using personality traits as metadata so that AI agents can personify these warriors in battles, with players betting on outcomes for profit.
  • RANN: An extension of Warriors AI-rena featuring a Training Center: An AI-backed on-chain platform that enables users to train and upgrade their existing warriors, an NFT Marketplace: where users can buy/sell NFTs through on-chain auctions, and an on-chain Leaderboard.
Github Demo Video Website
CLAMM-DEX-Engine
Solidity Foundry
  • It is a CLAMM (Concentrated Liquidity Automated Market Maker) decentralized engine similar to Uniswap v3, built from scratch.
  • This CLAMM-DEX can generate its liquidity providers up to 400% higher returns on the same capital when compared to traditional AMM-DEX.
